# Project RAG (Spring 2024)


## Overview

This project uses PineCone and  LlamaIndex as a  vectorDB and piepline.

## Requirements 
- PyPDF2
- pdfminer
- argparse
- llama-index
- transformers
- pdfminer.six
- tokenizers
- torch
- pinecone-client
- numpy



## Usage Examples
- `pip install -r requirements.txt` 
- `python upload.py --pdf_file=example.pdf`
